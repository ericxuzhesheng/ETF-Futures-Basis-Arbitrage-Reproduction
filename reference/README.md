# Reference Reports

This directory stores the three sell-side reports used as the reproduction
anchors for the ETF / index-futures basis-arbitrage project.

| File | Source focus | How it is used in this repo |
| ---- | ------------ | --------------------------- |
| `东证期货-股指期货套利策略系列一：股指期货与ETF的基差套利策略-20231011.pdf` | Orient Futures index-futures / ETF basis arbitrage | Basis-convergence and mean-reversion signal framing, including premium/discount regime interpretation. |
| `华泰柏瑞沪深300ETF与股指期货套利机会分析.pdf` | Huatai-PB CSI 300 ETF and index-futures arbitrage opportunity | CSI 300 ETF execution opportunity, cost assumptions, and no-arbitrage band context. |
| `银河期货-ETF期现套利策略-20240424.pdf` | Galaxy Futures ETF cash-futures arbitrage strategy | ETF cash-futures strategy comparison, multi-variety composite benchmark, and report-number validation targets. |

The PDFs are research references only. Reproducible data and generated outputs
live under `data/`, `results/`, and `figures/`.
